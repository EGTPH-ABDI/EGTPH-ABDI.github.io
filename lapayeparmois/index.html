✏️ تعديل — راقب جلول

📖 إعداد Firebase — مرة واحدة (≈10 دقائق)

1. افتح console.firebase.google.com وسجّل بحساب Google تاعك.
2. اضغط Create a project → سمّه مثلاً egtph → عطّل Analytics → Create.
3. من القائمة: Build ← Realtime Database → Create Database → اختر Belgium (europe-west1) → Start in locked mode → Enable.
4. فوق الجدول راح تلقى رابطاً مثل:
https://egtph-pay-default-rtdb.europe-west1.firebasedatabase.app
انسخه — هذا هو رابط قاعدة البيانات.
5. افتح تبويب Rules والصق هذا بالضبط ثم Publish:
{
  "rules": {
    "egtph": {
      "$key": {
        ".read": true,
        ".write": true
      }
    }
  }
}
6. ارجع للبرنامج (في الجهازين): الصق الرابط + اخترع مفتاحاً سرياً طويلاً (هو كلمة السر — نفسه في الهاتف والحاسوب) → 💾 حفظ → 🔄 مزامنة الآن.
🔒 ملاحظة: الحماية تعتمد على سرية المفتاح — اجعله طويلاً وعشوائياً (20+ حرف ورقم) ولا تشاركه مع أحد.
